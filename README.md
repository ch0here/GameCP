# GameCP
 Install VPS Linux (Ubuntu, Centos) in GameCP,
   Commands Centos:
                   yum install nano
                   nano gcp.sh  (Add script gamecp to gcp.sh)
                   wget https://raw.githubusercontent.com/ch0here/GameCP/master/c1.sh
                   sh c1.sh
                   -Done.
   Comands Ubuntu:
                  apt-get install nano
                  nano gcp.sh  (Add script gamecp to gcp.sh)
                  wget https://raw.githubusercontent.com/ch0here/GameCP/master/c1.sh
                  sh c1.sh
                  -Done.
